            1.2 云计算概述

            # 云计算概述

            2006 年Google CEO Eric Emerson Schmidt 提出云计算（cloud compute）的概念

            随着新技术的发展，云计算对于我们而言也已经不是一个陌生的名词，不知不觉我们已经习惯了在云上学习、工作、生活，慢慢我们已经逐渐离不开云计算，随着信息技术水平的不断发展，云计算将会成为引领未来整个信息系统建设的主导者。

            其实很多时候并不是技术引领时代和发展，而是时代的发展需求选择了技术，云计算技术也是如此，并不是因为构成云计算的技术恰好成熟然后催生出云计算，而是因为移动通信技术的发展解放了人类社会的衣食住行、教育、金融等各种需求，以此诞生的一大批互联网企业需要这样的综合技术来优化互联网产品从开发、测试、部署到运行维护的流程，而云计算恰好提供了云计算。

            云计算是分布式计算、并行计算、效用计算、网络存储、虚拟化技术、负载均衡、网络技术等计算机和网络技术融合发展的产物，很多知名IT企业——比如Google、Amazon、IBM——对云计算技术的发展、落地、普及做出了贡献。

            # 计算机发展的几次变革

            时间 变革
            1970s 大型机时代到微型机时代的变革
            1990s 微型机到互联网时代的变革
            2000s 互联网时代到云计算时代的变革

            # 1970s：大型机时代到微型机时代

            20世纪70年代前期，在Multics计划的废墟之上偶然诞生了可用于小型机的Unics操作系统，Unics先后经过C语言改写内核，然后从贝尔实验室流入到高校学术界，同时期各个高校和Geek不断完善了Unics——比如打印机和最早的计算机网络。

            值得一提的是20世纪70年代末期，美国国防部高级研究计划局（DARPA）决定选用Unix来实现其Arpanet的网络协议栈——TCP/IP协议， 这个合同不仅是Unics计算机操作系统的重要拐点，对于以后TCP/IP真正成为整个工业界事实上的网络协议标准也起到了重要的作用。

            因此1970s计算机的变革之处在于：

            • 计算机从用于军事商业的大型机逐渐走向微型机；
            • 广泛适用于各种规模、规格（可能除了x86）、品牌计算机的操作系统——Unics发展起来；
            • 最具生命力的计算机OS（Unics）和规模最大的计算机网络（ARPANET）——开始同流；

            ☞传送门：详细介绍

            # 1990s：微型机到互联网时代

            出现在上世纪90年代初，是从微型机时代到互联网时代的变革，互联网将数以亿计的信息孤岛汇集成庞大的信息网络，极大地提供了人类沟通、共享和协作的效率，丰富了社交和娱乐生活。

            # 2000s：互联网时代到云计算时代

            在本世纪初，逐渐产生了从互联网时代到云计算时代的变革，云计算将IT基础设施转变为像水、电、煤气一样的社会公用基础设施，超级数据中心成为IT资源和服务的提供者，一体化的信息和运维平台的交付模式对IT界产生重要影响。

            # 个人计算机的困扰

            • 个人计算机更新换代快；
            • 应用程序频繁更新；
            • 电脑因为负载过重或硬件问题宕机，导致数据丢失；
            • 病毒软件泄露了重要的个人数据和隐私，造成经济和精神损失；
            • 很多软件购买、下载、安装后，只是需要临时使用或者使用频率很低；

            # 企业计算机的困扰

            • IT部门工作人员经常需要在各个办公室之间解决员工电脑系统、应用软件的错误；
            • 临时产生的计算需求——比如测试某个应用软件，完成后设备会处于闲置状态，不具备经济性；
            • 急需供应的计算需求——比如市场规模扩大或者经营业务扩张，审批资金、购买设备、安装平台会花费很多时间，不具备灵活性；

            # 互联网遇到的问题

            • 互联网上的数据量高速增长，导致了互联网数据处理能力的不足；
            • 互联网上存在着大量处于闲置状态的计算设备和存储资源；
            • 服务器更新换代速度加快，企业升级费用昂贵

            而云计算实现了资源和计算能力的分布式共享，能够很好地应对当前互联网数据量的高速增长的势头。
