个人简介

# 个人简介

目前南京邮电大学网络工程专业在读，有过 vue、golang web 的开发经验，Linux爱好者，目前在熟悉运维技术和云原生技术，2021年2月份开始备战考研

求抱大腿

如果有虚拟化技术、容器开发、运维开发、软件定义网络等方向的大佬，求带，愿意成为大腿的小挂件

目前在做的事情：

  • 系统学习一遍Linux：虽然用过 Linux的时间不算短，但更多还是 野路子，以玩乐的性质居多，最终还是要落实到生产和服务上
  • 高要求学习好专业课：相比较计软院物联网学院的课程进度会慢不少，目前计算机网络、汇编和微机、网络安全、数据库、机器学习理论，都是非常基础但也很重要的课程
  • 积累实际运维经验：一方面熟悉传统的运维方案，另一方面熟悉自动化运维工具的部署、阅读自动化运维工具的文档，比如 gitlab、kubernetes
  • 项目开发（造轮子）：不管考研还是工作，大概率深造/研究的方向是运维开发、云原生、云计算、虚拟化、服务网络，目前正在查看检阅书籍、代码、论文
  • 思考和研究：在2020 疫情和调整期间，我完善、突破了对自己与外界的认知，我希望拓展更深层次的思维来塑造自己更多的可能性，比如哲学思维、产品意识、营销理念、交流共情
  • 治理和性灵：用规则、秩序、驱动来治理自己可以让人免于庸碌，而性灵的滋养能够让凡夫俗子（本座是也）在滚滚红尘中始终发现真实的自己、不断实现与超越自己

# 1. 求学生涯

  • 2017年 9月 入学南京邮电大学现代邮政学院 “网络工程（邮政工程）” 专业
  • 2018年 8月 转专业 ☞ 南京邮电大学物联网学院 “网络工程专业”
  • 2020年 1月 因个人原因休学半年（学业、职业规划，身体状态调整）
  • 2020年 8月 复学
  • 目前就读 南京邮电大学 ➜ 物联网学院 ➜ 网络工程 ➜ 2018级大三

# 2. 经历

  1. Linux 爱好者，进入大学之后，Linux 贡献了我大部分多巴胺分泌的来源，主要技术轨迹亦是以 Linux & 网络 为主干
  2. 每一个Linux 爱好者最后都会拥有一个 Mac，我也并不例外，Mac Mini 很香 😂

# 2.1 在校做过的编程经历

  • 程序设计周：基于 Sciter 引擎使用 html前端 和 C语言后端/存储 的学生综合测评系统
  • Java 大作业：基于 JavaFx 的富文本编辑器
  • 数据结构设计：基于二叉树表示的算术表达式的操作
  • 安卓移动开发：使用Flutter 技术实现图书馆订阅系统
  • 网络安全课程设计：使用golang 实现局域网监控和管理系统

# 2.2 社团经历和荣誉证书

暂时不谈，时间比较紧张。

名可名，非常名。

  • 社团存在的意义是提供一个平台让志趣相投的人聚集在一起，相互交流碰撞出火花，同时薪薪之火代代相承；
  • 比赛的最原始的意义是在为了竞赛奋斗的过程中，自我激励、管理、驱动，获得（可能有的）结果和荣誉的过程中，真正学习和拓展知识与技能；
  • 当然比赛在考核专业能力的功能性上，大部分时候的确是相对公正、可量化的手段，无可厚非

# 2.3 学术经历

有一段不太愉悦和光彩和经历，只能说能够理解当前科研的现状，也能体会到绝大多数科研和学术人员做的并不如同戏剧中那样起到变革和突破，大部分科研工作者做的事情仅仅是小修小补，或者实际应用、横向展开

尽管不算外行，依旧对科研抱有一些希望和热情，但更希望的是可以带着问题的心态去研究和寻找解决方案，也就是说更倾向于和工业界建立更紧密的联系，我自认为很难成为纯粹的 Researcher —— 并非不能和光同尘，只是暂时我还很难在长久找不到落地场景的工作中取得获得感、愉悦感，更不谈高效工作、落实输出

# 3. 一些务虚的方面

经历过很多之后，现在这个阶段，如果还有机会的话，在保证完成优先级更高的事情的前提下，依旧会尽力而为给自己找一条“小道” 与所谓 “捷径”

但更多希望得到的还是自己给自己的安全感 —— 自我治理自我驱动、独立判断的能力、提高自我认知层次、掌握科学的思维方法

在生存与独立的同时培育想象力、创造力、同理心、共情能力 —— 性灵
