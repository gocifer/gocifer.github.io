# 1. Turning

转折与决策

出生家庭、选择学校、确定工作、寻找伴侣，生命中所谓的大事其实就这么几件，很多时候人往往会在一些 “小节” 上犹疑不定，而对于这些极为重要的决策却非常草率。当然，当真的面临到必须做这些决策，可能绝大多数时候选择也有限，更加并不由自己。

但除了这些重大的事件显性的构建了命运轨迹，在生命中绝大多数时候，我们经历的仅仅是一些微小而不起眼的选择序列。

生过的一次小病、走过的一次旅行、听过的一次讲座、吃过的一顿美食、看过的一场电影；又或许仅仅是上课时与老师的一次共情交互、一次安静独处的放空发呆、一次荷尔蒙心动后的夜不能寐……

这些都可能成为我们人生极为重要的转折点，而当这些转折点出现的时候，我们一般都是并不为所知的。在做决策的时候我们往往并不能完整、准确的感知其对于未来生活、生命、情境的价值，人生由看似无尽的决策序列构建而成，但当真正经历的时候，其实有限而短暂。

下面是在我 turning point 起到重要作用，或者其本身成为我的重要转折点的链接

关注到Wonz学长的经历

最初认识到学长的缘由并不光彩 —— 南邮汇编实验课上宿友在群里发的“参考链接”，当时稍微关注了一下，万一后面不急之需可能还会用到学长其他的实验报告。

后来一再接触到和这位学长相关，固然缘分妙不可言，搜索算法极其优秀，其实更多还是学长足够优秀，在很多地方都留下了足够让人注目的足迹：

  • 因为个人原因在大三上学期期末结束后休学了半年，寒假期间才开始为自己做考研的打算（其实不算早了），当时比较关注的是南大计算机，在搜集资料的时候有看到学长的经历分享
  • 后来在世界上最大的同性交友网站“Github” 通过同在青柚工作室的同学 star 『Wonz』 学长的 “南邮黄页” 项目，真的是足够有心了
  • 也因此 fellow 了『Wonz』学长的Github 账号，通过他 “众多博客” 😂 了解到学长是我会让我关注的那一种人——细心、文艺、爱好、技术

很多时候人们对自己爱/求而不得的东西往往会付出更多关注，自驱执行、爱好广泛、技术专精、努力善良……这些属性是我一直渴望获得，但由于自我治理能力、自我驱动能力或者还有一些方法和外在因素的影响而未能获得。

这就是我一直在寻找的

我是一个没有安全感的人，我是一个想法很多的人，我是一个忘性很大的人……

人的大脑就像计算机，同一时间的信息处理能力取决于CPU处理器（智力）、内存（记忆力）、图形渲git染器（想象力）。

如同计算机一样，对于人类这些也都是有限的资源，大脑🧠 自带调度与保护机制，当我们在处理众多紧急、短期的任务的时候，大脑会把那些它认为不重要、不紧急的思绪忘掉或者藏入更深层。

固然，这样的机制让我们在社会与集体中成为一个“高效”、“务实”、“实用”的螺丝钉（打工人），但对于个体其实很难真正沉淀出专业能力、自我认识、多元思维，至多也仅仅是时间、人力堆砌的所谓经验罢了

我一直在寻找，我试图以一种（数据）安全可控、表现力强、自由开发扩展、编辑设计一致、多层次管理的方式，来表达、组织、管理、传播我的经历、认知和知识：

  • Onenote、印象笔记、Word 使用私有软件和数据格式，对于单个文档作业、课程笔记、项目白板来说是不错的选择，但非我所求；
  • html 虽然能够几乎满足我所有的要求，但为了编辑、样式、排版付出的成本过高，编辑可视性也并不友好；
  • 折中的方案是使用简化的 markdown 标记语法，正则转换或者编译成 html，可是主流编辑器如：印象笔记、Typora、Vnote 虽然编辑的功能和交互非常友好，但是拓展性、表达力、自由度、组织管理方便很难让我保持持续的愉悦感；
  • 直到 vuepress 和 Vdoing theme 的出现让我眼前一亮
    • 兼容主流 MD 语法
    • 丰富的 plugin，优化渲染、样式、排版、SEO
    • 可以发布为个人网站自由部署传播，
    • 分类、tag、知识库目录页、网页多级顶栏、archive，让我在基于文档属性聚合与分离、基于知识库、基于网站表现、基于时间线这样的维度来管理
    • 自定义 Vue 组件，
    • 自由开发、组织、扩展
    • github actions 持续集成/部署（CI/CD）

这是我在自由、独立、可控、表达、传播、扩展 各个层面取得的最佳平衡方案，这里感谢 @vuepress (opens new window) 项目组 和 @Evan Xu (opens new window)的@vuepress-theme-vdoing (opens new window)主题

另外值得一提的是徐大佬让我对于前端的追求，从仅仅使用 angular 写一些控制台、管理系统 提高使用vue自定义开发组件 🐶 :手动狗头:
