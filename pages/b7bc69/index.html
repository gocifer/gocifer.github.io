# shell概述

# shell是什么？

  • shell是一个命令行解释器，它为用户提供了一个向linux内核发送请求以便运行程序的界面系统级程序，用户可以用shell来启动、挂起、停止甚至是编写一些程序。
  • shell还是一个功能强大的编程语言，易编写、调试，灵活性强。shell是解释执行的脚本语言，在shell中可以直接调用linux系统命令。

# shell分类

  • Bourne shell:from 1979 used in unix,main filename---sh
  • C shell:mainly used in bsd,like C language

注:C shell 与 B shell 语法不兼容

  • Bourne shell家族：sh,ksh,bash,psh,zsh
  • C shell家族：csh,tcsh

Bash与sh兼容，linux使用其作为用户标准shell

# bash变量

什么是变量？

  • 变量是计算机内存的存储单元，其中存放的值可以改变
  • 可以用来存放字符串，方便系统引用它来获知计算机的相关设置

变量设置规则

  • 变量名称可以由字母、数字、下划线组成。但是不能以数字开头
  • 在bash中，变量的默认类型都是字符串型，如果要进行数值运算，则必须指定变量类型为数值型
  • 变量用等号连接值，等号两边不能由空格
  • 变量的值有空格，需要使用单引号或双引号包括
  • 在变量的值中，可以使用转义字符
  • 如果需要增加变量的值，那么可以进行变量值的叠加，不过变量需要用双引号包含"$变量名" 或用${变量名}包含
  • 环境变量建议大写，便于区分系统命令和bash环境变量
  • 如果是把命令的结果作为变量值赋予变量，则需要使用反引号或$()包含命令

变量处理

  • 调用：echo $name
  • 查看：set
  • 删除：unset name

# 其他配置文件和登录信息

  • ~/.bash_logout 注销时生效的环境变量配置文件

  • ~/.bash_history 历史命令保存文件

shell登录信息

  • /etc/issue 针对本地终端生效
转义符 作用
\d 显示当前系统日期
\s 显示操作系统名称
\l 显示登录的终端号
\m 显示硬件体系结构
\n 显示主机名
\o 显示域名
\r 显示内核版本
\t 显示当前系统时间
\u 显示当前登录用户的序列号

  • /etc/issue.net

    • 针对远程终端生效
    • 转义符在/etc/issue.net文件中不能使用
    • 默认该文件不会生效，需要在ssh的配置文件/etc/ssh/sshd_config中加入"Banner /etc/issue.net"(记得重启ssh服务)

  • /etc/motd

    • 不管是本地登录，还是远程登录，都可以显示此登录信息
