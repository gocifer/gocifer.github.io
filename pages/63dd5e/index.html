# 文件处理

# 命令格式与目录处理ls

ls [-选项] [参数]

  • 执行权限：所有用户
  • 功能：查看目录文件
  • 选项
    • -a --all，显示结果包括隐藏文件
    • -l 详细信息显示
    • -d 查看目录本身属性
    • -h 人性化显示
    • -k/m/g 以相应单位显示文件大小
    • -i 查看文件ID节点
  • 参数：目录路径

文件类型和权限 引用计数 所有者 所数组 文件大小(字节) 文件最后修改时间 文件名

d:目录,-:文件,l:软链接

u:所有者，g:所数组，o:其他人

r读，w写，x执行

一个文件只能有一个所属者和所数组

# 其它目录处理

mkdir [-p] 目录名

  • 功能：创建目录
  • 选项：
    • -p:递归创建

可以创建多个目录mkdir A B

cd

  • 功能：切换目录
  • 参数：“.” 表示当前目录，“..”表示上级目录

rmdir

  • 功能：删除空目录

pwd

  • 功能：显示当前路径

cp [选项] 源对象1 [源对象2] 目标对象

  • 功能：复制
  • 选项：
    • -r：递归，用于复制文件夹
    • -p：保留文件属性

可以复制的同时完成易名

mv [-rf] 源 目标

  • 功能：移动、改名

rm [-rf] 文件或目录

  • 功能：删除文件
  • 选项：
    • -r：删除目录(递归删除文件夹及其子文件和目录)
    • -f：强制删除，不确认

# 文件处理

touch

  • 功能：创建文件

使用特殊字符命名文件：1.“”之内添加特殊字符 2.使用转义字符\

eg:touch programe\ files

cat

  • 功能：查看文件内容
  • 选项：
    • -n：显示行号

tac

  • 功能：倒着显示文件内容

more 文件名

  • 功能：分页显示文件
  • 空格或f：翻页
  • （enter）：换行
  • q或Q：退出

less 文件名

  • 功能：分页显示，上翻，搜索
  • ：/搜索（类vim）
  • pageup：上翻一页
  • 上箭头：向上翻一行

head [-n 数字] 文件名

  • 功能：显示前几行，默认前10行
  • 选项：
    • [-n 数字]：指定查看行数

tail 选项 文件名

  • 功能：显示末尾几行，默认前10行
  • 选项：
    • [-n 数字]：指定查看行数
    • -f：动态显示
tail -f /var/log/messages

# 链接

ln [-s] 源文件 目标地址

  • referred:link
  • locate:/bin/ln
  • 选项：
    • -s：生成软链接

软链接：lrwxrwxrwx，类似windows快捷方式,文件非常小，箭头指向源文件；软链接可以指向目录。

硬链接：实际上是将文件指向文件索引节点，不会另外占用实际空间，和源文件共用存储的文件i节点；文件保持同步；不能跨分区；不能针对目录使用

一个文件一定有一个i节点，但是一个i节点不一定只有一个文件

文件被彻底删除的标志是该文件的索引节点不再指向任何文件

# 更改文件权限

chmod {ugoa}{+-=}{rwx} 文件或目录

chmod 755 文件或目录

  • referred：change the permissions mode of a file
  • locate:/bin/chmod
  • permissions:users or root
  • 功能：改变文件或目录权限
  • 选项：
    • -R：递归修改
chmod g+w,o-r aa.list
chmod g=rw aa.list

chmod 755
1
2
3
4

权限位：rwx：421 根据进制位判断相应权限大小

修改某一文件夹权限并不会更改目录下面的文件的权限

# 文件权限详解

代表字符 权限 对文件含义 对目录含义
r 读 查看文件内容 列出目录内容
w 写 可以修改文件内容 可以在目录中创建、删除文件
x 执行 可以执行文件 可以进入目录

注意：可以删除文件的权限要求不是对文件本身有w权限，而是对文件所在父目录有w权限

注意：往往对于文件目录r和x成对出现（755），即往往进入和查看权限同时出现

# 文件所有权

chown owner[:group] file

  • referred：change file ownership
  • locate:/bin/chown
  • permissions:root
  • 功能：改变文件所有者和所属组
  • 选项：
    • -R：递归修改

chgrp [group] [file]

  • referred：change file group ownership
  • locate:/bin/chgrp
  • permissions:root
  • 功能：改变文件或目录所属组
  • 选项：
    • -R：递归修改

umask [-S]

  • referred：the user file-creation mask
  • locate:shell内置
  • permissions:root
  • 功能：显示、设置文件的缺省权限
  • 选项：
    • -S：以rwx显示文件的缺省权限

在linux系统中，默认新建的文件没有可执行权限

umask默认显示的权限掩码

umask -S
umask 023   //设置默认权限为754

# 文件搜索

# find [搜索范围] [匹配条件]

  • locate：/bin/find
  • permissions:owner
  • 功能描述：文件搜索
  • 选项：
    • -name 名字：指定文件名，精准搜索
    • 正则通配符*?：模糊搜索
    • -iname：不区分大小写
    • -size :+n大于；-n小于；n等于（数据块计算）
    • -user 用户
    • -group 组名
    • -amin：访问时间access
    • -cmin：文件属性change
    • -mmin：文件内容modify
    • -type：根据文件类型：f文件；d目录；l软链接
    • -inum：根据i节点查找
    • -a：前后两个条件同时满足
    • -o：两个条件满足一个即可
    • -exec/ok [command] {}\：对结果进行操作，-ok会对操作依次询问确认
find /etc/ -iname *init??
find /etc/ -cmin -5
find /rtc/ -size +1638 -a -type f
find /etc/ -name inittab -exec ls -l {}\;
find . -inum 33333 -exec rm {}\;

# locate filename

  • locate：/usr/bin/locate
  • permmission：users
  • 功能：在文件资料库中查找文件
  • 选项：
    • -i：忽视大小写

locate会定期更新文件资料库（位于/var/lib/mlocate/mlocate.db）

缺陷：locate并不是实时更新资料库，所以有时需要手动更新资料库updatedb,locate不会收录/tmp目录下的文件

# gzip

gzip [file]

  • locate:/bin/gzip
  • 功能：压缩
  • 选项：
    • -d：解压缩
    • -c：保留源文件

注：

  • gzip命令只能压缩文件
  • gzip压缩完后不保留源文件

gunzip [file.gz]

  • locate:/bin/gunzip
  • 功能：解压缩
  • 选项：
    • -c：保留源文件

注：利用重定向压缩，解压到指定目录

gzip -c chrome.rpm >/tem/chrome.gz
gunzip -c chrome.gz > /tem/

# tar

tar [选项] 压缩后的文件名 目录

  • locate：/bin/tar
  • 功能：打包目录和压缩文件
  • 选项：
    • -v：显示详细信息
    • -f：指定文件名
    • -c：打包（tar）
    • -z：打包同时以gzip压缩（tar.gz）
    • -j：打包同时以bzip2压缩（tar.bz2）

tar [选项] 压缩包

  • 功能：解包和解压缩
  • 选项：
    • -v：显示详细信息
    • -f：指定文件名
    • -x：解包
    • -z：解压缩gzip压缩包
    • -j：解压缩bzip2压缩包
    • -C：解压到指定目录

tar -ztvf 压缩包

  • 功能：查看压缩包中的文件目录

# zip

zip [选项] [压缩后的文件名] [文件或目录]

  • locate：/usr/bin/zip
  • 功能：压缩
  • 扩展名：zip
  • 选项：

unzip [压缩文件]

  • locate:/usr/bin/unzip

unzip chrome.zip -d /tem/

  • 法1，利用pyton来处理
#!/usr/bin/env python
# -*- coding: utf-8 -*-
# uzip.py

import os
import sys
import zipfile

print "Processing File " + sys.argv[1]

file=zipfile.ZipFile(sys.argv[1],"r");
for name in file.namelist():
    utf8name=name.decode('gbk')
    print "Extracting " + utf8name
    pathname = os.path.dirname(utf8name)
    if not os.path.exists(pathname) and pathname!= "":
        os.makedirs(pathname)
    data = file.read(name)
    if not os.path.exists(utf8name):
        fo = open(utf8name, "w")
        fo.write(data)
        fo.close
file.close()

# 网络命令

# write 用户名

  • locate：/usr/bin/write
  • permission：all user
  • function：给用户发信息，Ctrl-D保存结束

注：只有当对方在线消息才可以发送出去

# wall [messages]

  • referred：write all
  • locate：/usr/bin/wall
  • function：发送广播信息
wall hello world
1

# ping 选项 ip

  • locate：/bin/ping
  • permission：all
  • function：测试网络连通性
  • 选项：
    • -c：指定发送次数

注意：windows中默认只会发送四次数据报，而linux中则会一直ping下去

# ifconfig 网卡名称 ip地址

  • referred：interface configure
  • locate：/sbin/ifconfig
  • permission：root
  • function：查看和设置网卡信息
ifconfig    //直接查看所有网卡信息
ifconfig rth0 192.168.61.2
ifconfig ens32 192.168.61.3

# last

locate：/usr/bin/last permission：all function：查看当前和过去登入系统的用户信息

# lastlog [-u uid]

locate：/usr/bin/lastlog permission：all function：检查用户最后一次登录的时间

# traceroute address

locate：/bin/traceroute permissions：all function：显示数据包到主机间的路径

traceroute www.baidu.com
1

# netstat [选项]

  • locate：/bin/netstat
  • permission：all
  • function：显示网络相关信息
  • options：
    • -t：TCP协议
    • -u：UDP协议
    • -l：侦听
    • -r：路由
    • -n：显示ip地址和端口号
netstat -tlun       //查看本机监听的端口
netstat -an         //查看本机所有网络连接
netstat -rn         //查看本机路由表
