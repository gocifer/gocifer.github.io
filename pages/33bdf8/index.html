gnome_extensions

# gnome_extensions

# Gnome桌面环境常用扩展

  • autohide topbar

  • Docker Integration A status menu for managing docker containers.

  • hidetopbar

  • Coverflow Alt-Tab 功能与AlternateTab相同，但提供了一个更酷炫的界面

  • Dash to Dock 不解释

  • no title bar 把三键集合到顶栏

  • hide title bar

  • TopIcons Plus 顶栏显示应用图标，输入法切换的时候特别有用

  • Dynamic Panel Transparency by rockon999

  • Walkpaper by blinkbp ONOFF: Set different wallpaper for each workspace by having the wallpaper switch on workspace change.

  • Removable Drive Menu

  • Dynamic Panel

  • Night Light Slide

  • Simple Dock Gnome Dock 配置工具，能够配置 dock 的位置、透明度等

  • ToDo.txt 在右上角增加一个可以快速添加「待办事项」的按钮，一个简单的 To-do-list

  • Remove Dropdown Arrows

  • redshift redshift 插件提供了设置屏幕日夜色温的图形接口，能改在夜晚减轻屏幕对眼睛的刺激。参见为 Linux 启用色温和亮度调节工具。

  • media player indicator 在界面上放置一个音乐播放指示器：可以将位置设置在系统菜单、Topbar 的中部和右部。

  • Web Search Dialog 提供热键，可以在 gnome-shell 中快速进行 Web 搜索：

  • Hide Clock by grantmasterflash

  • Appfolders Management extension

  • gnome-pomodoro

  • Google Earth Wallpaper

  • Bing Wallpaper Changer

  • Tilix

  • Places Status Indicator 为一些常用目录提供快速入口

  • system-monitor (by Cerin)顶栏显示CPU、内存、网速、温度、电池电量等信息，安装前需要解决依赖，在Debian上方法如下$ sudo aptitude install gir1.2-gtop-2.0 gir1.2-networkmanager-1.0(如果你觉得它依赖不好解决，也可以使用另外一个具有相似功能的System Monitor)

  • EasyScreenCast 屏幕录像，做演示时比较方便

  • User Themes 从本地安装shell主题

  • Dynamic Top Bar利用Dynamic Top Bar,顶栏会在不存在最大化窗口时以透明方式显示。而在对窗口进行最大化后，顶栏会重新出现供我们使用。

  • NetSpeed 顶栏显示网速，喜欢简洁的Simple net speed也非常不错

  • Applications Menu 顶栏显示应用菜单，我比较少用，但刚从Windows切换过来的或许需要

  • Workspace Indicator 在顶栏显示当前所处的工作空间

  • AlternateTab Alt+Tab切换应用

  • Wikipedia Search Provider如果大家像我一样经常查询维基百科，那么请尝试这项扩展。安装之后，点击Activities按钮而后输入大家需要查询的条目名称即可。举例来说，要查询Linux的相关内容，点击该按钮而后输入w linux即可显示相关结果。

  • Auto Move Windows 当应用创建窗口时自动移动到某个特定的工作空间

  • Easy Screen Cast在GNOME桌面中进行视频/音频截取

  • Caffeine 取消自动锁

  • Clipboard Indicator 剪贴板管理

  • EasyScreenCast 屏幕录像，做演示时比较方便

  • Gnomodoro 简易番茄钟

  • Pomodoro 增强的番茄钟，推荐使用。严格来说，它已经不是扩展了，作者现在将它做成了一个本地应用。对于Debian，新的安装方法如下$ sudo aptitude install gnome-shell-pomodoro

  • Lock Keys 顶栏显示Numlock和Capslock的状态

  • OpenWeather 顶栏显示天气情况，gnome自带的weather不支持顶栏显示，用起来不方便

  • TaskBar 类似于Windows任务栏，提供了可选的预览功能，可定制选项多

  • Todo.txt GTD应用，顶栏显示比较醒目

  • Window List 在屏幕底部显示窗口列表，提供类似于Windows任务栏的功能，习惯Windows的可以考虑用这个过渡一下
